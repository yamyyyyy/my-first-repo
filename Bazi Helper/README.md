# 八字命理智能分析助手

基于25部经典命理古籍开发的智能八字分析助手，支持多流派综合分析。

## 功能特性

- **八字排盘**：输入出生日期自动排盘，支持四柱、大运、流年计算
- **五行分析**：分析五行强弱、喜忌用神、格局判定
- **性格分析**：基于十神和五行分析人物性格特征
- **感情分析**：分析桃花、配偶星、正缘时机、婚姻运势
- **事业分析**：分析天赋特长、适合行业、财运趋势
- **多流派分析**：子平、禄命、纳音、盲派、调候五大流派综合分析

## 技术栈

- Node.js
- 纯JavaScript实现，无需外部依赖

## 安装运行

```bash
# 安装依赖
npm install

# 启动CLI界面
npm start

# 或直接运行
node src/cli/baziCli.js
```

## 使用说明

1. 选择"排盘"输入出生日期（年、月、日、时）和性别
2. 选择需要的分析功能（五行、性格、感情、事业、流派）
3. 或选择"完整"生成完整的命理分析报告

## 支持的流派

1. **子平学派**：日主强弱、格局判定、用神选取
2. **禄命古法**：禄、命、身三奇，纳音五行
3. **纳音学说**：六十甲子纳音，音声相合
4. **盲派命理**：神煞、刑冲合害、象法直断
5. **调候学派**：四季气候，调候用神

## 注意事项

- 结果仅供参考，请勿过分迷信
- 时间使用24小时制
- 出生时间以当地真太阳时为准
